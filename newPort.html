Minecraft RAT Scanner
A sophisticated security tool that leverages machine learning to detect Remote Access Trojans (RATs) and malicious code in Minecraft JAR files. This project was born from a passion to protect the Hypixel Skyblock community, where users frequently download quality-of-life mods that unfortunately often contain embedded RATs targeting players' accounts and personal information.

Key Features:

  • ML-based detection system for identifying malicious patterns
  • Automated feature extraction from JAR file structures
  • Safe mod verification and whitelisting capabilities
  • Comprehensive scan reporting and logging
  • Custom training dataset support for continuous improvement
  • Isolated execution environment for security testing

Security Impact & Motivation:

This project was developed to help users defend against RATs in a gamemode in Minecraft called Hypixel Skyblock. In this community, users download mods to help with quality of life features. However, in this community there is known to be RATs embedded in the mods. Therefore, I developed this project to help defend against this threat and protect fellow community members.

This project demonstrates understanding of malware analysis techniques, machine learning applications in cybersecurity, and the importance of protecting gaming communities from sophisticated threats. The tool provides automated threat detection capabilities that can scale to analyze thousands of mods efficiently.

Man-in-the-Middle Attack via ARP Poisoning
A hands-on demonstration of a man-in-the-middle (MITM) attack using ARP poisoning in a controlled lab environment. This project showcases the interception of network traffic between a mobile phone and a router on an unmanaged switch, highlighting the vulnerabilities of the ARP protocol.

Key Features:

  • ARP table manipulation and spoofing techniques
  • Network traffic interception and analysis
  • Real-time packet capture and inspection
  • Demonstration of protocol vulnerabilities
  • Controlled lab environment testing
  • Educational documentation and analysis

Attack Scenario:

In this scenario, I targeted a mobile phone and a router within a known lab environment. By sending spoofed ARP messages, my machine was able to intercept all traffic between the two devices, effectively becoming the man-in-the-middle.

Learning Objectives:

This project demonstrates a practical understanding of how ARP poisoning works and how it can be used to compromise a network. It also highlights the importance of using secure protocols like HTTPS to protect against this type of attack.

Technologies Used:

Python Scapy Wireshark Network Analysis ARP Protocol Packet Manipulation
